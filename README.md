# Bone Cancer Detection using Artificial Neural Networks (ANN)

This project uses a Deep Learning model (ANN) to analyze data and detect the presence of bone tumors.

## Project Structure
- `app.py`: Flask web server.
- `models/`: Contains the trained `.h5` ANN model.
- `templates/`: HTML interface for the user.
- `Bonetumor.csv`: Dataset used for the project.

## How to Run
The app is built with Flask and TensorFlow.
